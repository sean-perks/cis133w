Plot Monumentation

Return to Top

Plots were marked with 3/8-inch by 1-foot rebar, capped with a green rubber safety cap, at each vertex of the polygon. These markers are buried in vegetation and difficult to find. There are also several missing points that must be re-monumented using high precision GPS according to the nectar_plot_alternative and violet_plot_alternative polygons on the OSB Habitat Enhancement ArcGIS Online (AGOL) web map or Field Maps app.

Task 1: Locate and Verify Monumentation

Summary: Locate all plot points to find and verify monumentation exists. Monument all missing plot points.

Method:

  1. Check the OSB Habitat Enhancement map to see the locations of known missing monuments represented by orange circles in the Plot Monumentation layer.
  2. Using the highest precision GPS device available, locate where the plot points are and look for the rebar monument.
  3. If the rebar is missing:
    1. Monument by hammering in rebar and capping with a green cap. If the plot was known to be missing, delete the Plot Monumentation layer in Field Maps to show the plot has been found/re-monumented.
    2. Note: If the monument was not known to be missing and still cannot be found, grab a buddy to check your work then go ahead and re-monument.

Photo Points

Photo Points qualitatively track vegetation changes over time. Permanent photo points would be established on four corners of the Enhancement Area that would capture different views of the Nectar Buffer and Violet Plots. The location and direction of each photo point would be recorded using a handheld GPS device. Photo point locations and photos would be included in the annual report. Photo points would be collected annually between June and August or when vegetation surveys are conducted.

Task 1: Determine Photo Point Locations

Summary: Photo points will be taken at some of the vertices of the enhancement plots to track progress over time. Look at the OSB Habitat Enhancement ArcGIS Online (AGOL) web map or Field Maps app to determine where the photo points should be taken. These will be used for all subsequent years. Not every vertex gets a photo point.

Method:

  1. Determine where the photo points should be based on the number of vertices of the enhancement areas it takes to visually cover most of the area. For example, a square enhancement plot would likely have a photo point taken at each corner looking inward. For irregular shaped enhancement plots, some corners may be skipped. An empty point may be taken while planning and can be edited when the point is visited.

Task 2: Capture Photo Points

Summary: Take a photo point at each of the chosen locations.

Method:

  1. Fill out the Photo Point feature on the OSB Habitat AGOL web map or Field Maps app. Record the azimuth and take the photo as an attachment to the point. If the point already exists, use Field Maps to open the point and edit the feature.

9.1.1 Nectar Habitat Monitoring

OSBHEP Summary: During the baseline assessment, 1-square-meter quadrats would be randomly dispersed throughout the Nectar Buffer area. Quadrat locations would be temporarily marked with pin flags in the northwest corner of the quadrat. Each quadrat would be assigned a number, and the location and number of each quadrat would be recorded using a handheld GPS device.

Data Collected:

  1. Species code and area cover for each species.
  2. Area cover for bare ground, rock, and thatch.
  3. Number of nectar plant flowering stems and early blue violet individuals would be recorded for each quadrat.

The number of quadrats surveyed will depend on site variability; a minimum of 30 non-permanent quadrats would be surveyed initially. Additional quadrats will be assessed until the cumulative running average of the number of nectar plant flowering stems per quadrat varies by less than 5%. This criterion will be applied for each site visit. The average density of nectar plant flowering stems in the Nectar Buffer would be calculated by summing the total number of nectar plant flowering stems within each quadrat, summing the total for all quadrats, and dividing by the total number of quadrats. The same methodology would be followed in subsequent monitoring years. Target nectar plant density would be achieved if the density is at least five flowering stems/m² in at least 66 percent of quadrats surveyed.

Task 1: Locate and Prepare Plot

Summary:Thirty Enhancement Plot Survey Points (survey_point_enhancement) have been randomly placed in the enhancement areas on the OSB Habitat Enhancement Field Maps App. Find a point, set up the plot, and open the plot point to edit and begin recording data.

Method:

  1. Using foldable meter sticks, or some kind of 1 meter x 1 meter square, set the 1 meter x 1 meter plot roughly centered on the AGOL plot point. If a plot intersects something undesirable, step 5 paces in a suitable direction and set the plot there. Click on the survey_point_enhancement point you are currently at. Click “edit” to open the point data form.

Task 2: Measure Plot

Summary:Record data onto the Enhancement Plot Survey Points (survey_point_enhancement) in the OSB Habitat Enhancement Field Maps App.

CAUTION: When editing a point, it is possible to accidentally MOVE the point. Please avoid this.

Method:

At each survey_point_enhancement, take the following data:

  1. Species code and area cover for each species (estimated as a percentage 1-100%). Ideally, two people confer on the percent cover.
  2. Area cover for bare ground, rock, and thatch (1-100%).
  3. Number of nectar plant flowering stems and early blue violet individuals would be recorded for each quadrat.

9.1.2 Early Blue Violet Surveys

OSBHEP Summary:

Early blue violets would be counted within the Violet Plots. In patches where all individuals can be efficiently counted, a complete census would be performed, and the number of early blue violet individuals would be recorded for each Violet Plot. In larger, denser early blue violet patches, a sub-sampling approach may be utilized. To perform the subsampling, 10 randomly located 0.5-meter-wide belt transects would be run through each Violet Plot and the number of early blue violet individuals counted in each transect. The results of the first 10 transects in each Violet Plot would be used to calculate the number of additional transects necessary to determine an estimate of the number of early blue violets within the desired margin of error (10 percent). If necessary, additional belt transects would be surveyed until the standard error is less than 10 percent of the mean. The average early blue violet density would be calculated by summing the individual counts in each transect and dividing that number by the sum of the transect areas. An early blue violet individual would be defined as a plant covering a 4-centimeter (cm) x 4 cm area. This area-based definition aids in determining an individual in dense patches. If a plant is growing by itself and appears to be a single individual, it would still be assessed based on this definition; in this case, what may be just a single individual would be counted as multiple individuals if it was notably larger than 4 cm x 4 cm (Silvernail and Bahm 2020).
